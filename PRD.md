# PRD - Solocal AppSec

## Vision
Automatiser des verifications AppSec courantes avec des outils legers.

## Probleme
Les audits AppSec sont couteux et peu frequents.

## Utilisateurs
- AppSec engineers
- Auditeurs securite

## MVP (fonctionnalites)
- OSINT sous-domaines (offline)
- Decode JWT
- Anti-scraping (rate limiting demo)
- Detecteur SQLi (offline)
- Analyse logs web

## Evolutions
- Rapport HTML des controles
- Exports JSON standardises

## KPI
- Vulns detectees
- Temps d'audit

## Entrees / Sorties
- Entrees: URLs, tokens, logs
- Sorties: logs/rapports

## Contraintes
- Demos sans reseau possibles

## Hors perimetre
- Scanner automatique complet OWASP
